# To-Do List App 📝

A simple and interactive To-Do List web application built with **HTML, CSS, and JavaScript**. This project allows users to add, manage, and organize daily tasks efficiently.

## Features ✨
* Add new tasks
* Mark tasks as completed
* Delete tasks from the list
* Input validation to prevent empty tasks
* Keyboard support (press **Enter** to add tasks)
* Simple and responsive design

## Technologies Used 🚀

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (ES6)** – Functionality and interactivity

## How to Use 📌
1. Type your task into the input field.
2. Click the **Add** button or press **Enter**.
3. Your task will appear in the list.
4. Click on a task to mark it as complete.
5. Use the delete option to remove tasks.

## Installation ⚙️
To run this project locally:

```bash
git clone https://github.com/your-username/todo-list.git
cd todo-list
```

Then open the `index.html` file in your browser.
